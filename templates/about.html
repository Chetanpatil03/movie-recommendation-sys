About Movie Recommender

Your personalized gateway to discovering amazing movies and web series

Project Overview

This Movie Recommendation System is a comprehensive web application built with Flask, designed to help users discover movies and web series across different categories including Hollywood, Bollywood, and Web Series. Using advanced content-based filtering algorithms with TF-IDF vectorization and cosine similarity, the system provides personalized recommendations based on movie plot similarities. Movie data is enriched using the OMDb API for posters and additional details.

Technologies Used

Backend
  • Flask Framework
  • Pandas for Data Processing
  • Scikit-learn (TF-IDF, Cosine Similarity)
  • TMDB API Integration
Frontend
  • HTML5 & CSS3
  • Bootstrap 5
  • JavaScript
  • Font Awesome Icons

Key Features

Smart Search

Search across all categories with instant results

Advanced Filtering

Filter by genre, rating, and category

Personalized Recommendations

AI-powered suggestions based on content similarity

Detailed Information

Cast, director, trailers, and IMDB links

How It Works

  1. Content Analysis: Movies are analyzed based on their overviews using TF-IDF vectorization
  2. Similarity Calculation: Cosine similarity measures how similar movies are to each other
  3. Recommendations: The system suggests movies with the highest similarity scores
  4. Enhanced Data: TMDB API enriches movie data with posters, cast, and additional details
